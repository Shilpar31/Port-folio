Q&A chatbot-RAG Project

This project is a PDF-based Q&A app using LangChain, Ollama, Streamlit, and ChromaDB. Users upload a PDF via Streamlit, which is split into chunks and embedded as vectors. These embeddings are stored in ChromaDB for similarity search. When a question is asked, the most relevant chunks are retrieved and passed to a local LLM via Ollama. The generated answer is then displayed to the user in the interface.
Technologies Used: Lang-chain, Ollama, Streamlit, Chromadb
